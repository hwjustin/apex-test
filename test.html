Where APEX plugs into this flow

In this demo, Prebid is calling test DSP endpoints directly. In the APEX architecture, you expose an OpenRTB-compliant SSP endpoint that Prebid calls as a bidder adapter. The publisher.id in each bid request maps to an ERC-8004 agent identity. The winning bid triggers an x402 payment — no burl pixel, no invoice reconciliation. Add bidder: 'apex' to any ad unit and your validator network handles the rest.
